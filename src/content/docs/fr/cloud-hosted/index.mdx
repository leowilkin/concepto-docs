Au lieu d'héberger Concepto AI sur votre propre serveur, vous pouvez l'héberger sur nos serveurs cloud. C'est une excellente option si vous n'avez pas de serveur ou si vous ne souhaitez pas en gérer un.

## Comment obtenir l'accès
Remplissez ce formulaire pour obtenir l'accès à l'hébergement cloud : https://tally.so/r/wLPkel
