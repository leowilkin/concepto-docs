**Auto-hébergé** est une excellente option si vous souhaitez avoir un contrôle total sur votre déploiement. Vous pouvez modifier le code à votre guise et utiliser votre propre domaine.

:::caution[Note]
Vous devez disposer d'un **serveur** et d'un accès de niveau **OpenAI Tier 2** pour héberger Concepto AI. Si vous n'en avez pas, nous vous recommandons d'utiliser la version hébergée sur le cloud.
<LinkCard title="Hébergé sur le cloud" href="/fr/cloud-hosted" />
:::

<Tabs>
  <TabItem label="Avantages">
  + Vous avez un contrôle total sur votre déploiement
  + Contrôle complet des tarifs, y compris les clés API
  + Vous pouvez utiliser votre propre domaine
  + Modifiez le code à votre guise
  </TabItem>
  <TabItem label="Inconvénients">
  - Besoin d'accéder à l'API et aux clés d'OpenAI
  - Vous devez disposer d'un serveur
  - Vous devez avoir certaines connaissances techniques
  - Vous devez avoir un domaine
  - Peut être plus cher que la version cloud
  </TabItem>
</Tabs>
