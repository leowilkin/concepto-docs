---
title: Comment fonctionne Concepto AI?
desc: Découvrez le fonctionnement de Concepto
---
1. Dessinez un croquis de votre idée pour votre site web à l'aide de l'ensemble d'outils de croquis intégré.
2. Concepto AI convertira automatiquement votre croquis de SVG à PNG.
3. n utilisant la dernière API GPT4-Vision d'OpenAI, Concepto AI convertira ensuite votre PNG en un fichier HTML et CSS Tailwind.
Un aperçu apparaîtra à l'écran du code et du site web.
Vous pouvez ensuite télécharger votre fichier HTML et CSS Tailwind et l'utiliser dans votre projet.
