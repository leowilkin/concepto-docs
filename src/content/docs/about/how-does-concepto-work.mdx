---
title: How does Concepto AI work?
desc: Learn how Concepto works
order: 2
---
1. Draw a sketch of your idea for your website using the inbuilt sketching tool suite.
2. Concepto AI will automatically convert your sketch from SVG to PNG.
3. Using OpenAI's latest `GPT4-Vision` API, Concepto AI will then convert your PNG into a HTML & Tailwind CSS file.
4. A preview will apper on the the screen of the code & the website.
5. You can then download your HTML & Tailwind CSS file and use it in your project.
